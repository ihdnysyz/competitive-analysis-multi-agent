# 🚀 Multi-Agent Competitive Intelligence Platform (MVP)
**多智能体协同自动化竞品动态情报网络与深度分析平台**

基于 [CrewAI](https://github.com/joaomdmoura/crewai) 和大语言模型（LLM）构建的企业级竞品分析自动化引擎。通过多个具备特定设定的 AI Agent 协同工作，实现从“情报抓取”到“市场分析”再到“战略制定”的端到端自动化。

## 👥 智能体团队架构

平台内置三个核心 Agent 角色，通过流水线（Sequential Process）协同工作：
1. 🕵️ **情报侦察员 (Intelligence Scout)**: 自动化检索全网竞品动态、新闻和产品迭代。
2. 📊 **市场分析师 (Market Analyst)**: 接收原始情报，清洗数据并产出 SWOT 深度分析矩阵。
3. 💡 **战略顾问 (Strategy Consultant)**: 基于分析报告，产出针对己方企业的 3 条可执行反击策略。
